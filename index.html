Cuadro de mando socioeconómico de Marbella

Panel de indicadores oficiales del municipio de Marbella (29069). Todos los datos son dinámicos: un proceso automático descarga cada día las fuentes oficiales (INE, IECA/Junta de Andalucía y SEPE) y las deja servidas junto a este panel, de modo que al abrirlo —hoy o dentro de un año— siempre muestra el último dato publicado, sin intervención manual.

Fuentes y método

Datos oficiales actualizados automáticamente. INE Tempus3IECA / BADEASEPE datos abiertos

El proceso de actualización (fetch_data.py) se ejecuta de forma programada en GitHub Actions y vuelca los datos a /data/*.json. Este panel los lee desde el mismo origen, por lo que no depende de que ninguna fuente tenga CORS ni de ningún PC encendido.
